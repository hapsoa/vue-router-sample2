WebStrom 메모리 사용 확장하는 법

cd /
cd Applications/WebStorm.app/Contents/bin 에 들아가서 

vi webstorm.vmoptions

에 들어가서 

1 -Xms512m
2 -Xmx4096m

로 변경한다.

하지만 WebStrom이 업데이트 되면 저 파일이 변경될 수 있기 때문에,

각 프로젝트마다 웹스톰 프로그램에서

Help -> Edit Custom VM Options ...
로 들어가서 위의 설정으로 변경한다.
